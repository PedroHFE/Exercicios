Tipos de links

Links externos

Para criar links usamos a tag A, é obrigatorio colocar o HREF do link.

O A da tag vem de Ancora e o HREF é de Referencia Hyper Texto.

Sempre que cololocar um link externo não esquecer de usar na tag A, o target _blank e o rel external.

O target é alvo, o _blank é para abrir uma nova janela em branco.

O rel mostra que o link é do tipo externo.

Para mais informções pode Verificar no GitHub a apostila dessa aula.

Veja também o video dessa aula clicando aqui.

Tipos de target são o _blank, _self, _parent, _top e o frame

  1. _blank vai abrir o link em nova janela em branco
  2. _self vai abrir o link na janela ou frame atual (padrão)
  3. _parent similar ao uso do _top em uma referência à janela mãe
  4. _top vai desfazer todos os frames e abrir o destino no navegador completo
  5. nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir

O uso de frames é uma técnica quase em desuso, o frame já é uma tag obsoleta.

Links Internos

Para os links Internos não precisa colocar a URL do site basta só o caminho da pagina, basta
usar o CTRL + espaço para selecionar a pasta, caso esteja dentro de outra pasta use ../ e você conseguira voltar.

Links para download

Para fazer os downloads além de você colocar o caminho em HREF, ira precisar colocar download e o caminho
também precisara informar o tipo do documento ex: type= "application/pdf".

Consultar a lista de Media Types para usar corretamente na hora de colocar no codigo.
Site para a lista é o Iana.
