Hierarquia de tabelas simples

Em primeiro lugar temos a tag table = tabela dentro de table temos, table row = linha de tabela, dentro do table row temos header = cabeçalho de tabela e table data = dado de tabela. O w3c diz que o fechamento das tags <td>(Table data) e <tr> (Table row) é opcional, vale lembra que se a hierarquia não for obedecida os dados vão aparecer antes ou depois da tabela.

A1 A2 A3
A2 B2 C2
A3 B3 C3

<table>
<tr>
<td>A1</td>
<td>A2</td>
<td>A3</td>
</tr>
<tr>
<td>A2</td>
<td>B2</td>
<td>C2</td>
</tr>
<tr>
<td>A3</td>
<td>B3</td>
<td>C3</td>
</tr>
</table>

Alinhamento da tabela

Para alinhar na horizontal, podemos usar a tag text-aling (left-center-right) e para alinhar na vertical, usamos a tag vertical-align(top-middle-bottom).

Coloque sempre o scope(escopo) das th(titulos), scope="col" para os dados que ficarão na coluna (a baixo), e scope="row" para os dados na mesma linha.

Tags semanticas para tabelas grandes

Nas tabelas grandes ainda usamos as tags tr(table row) e td(table data) mas agora aprenderemos a anatomia das tabelas grandes. Sempre vai começar com a tag table mas dentro de table, usaremos a Thead(cabeça da tabela), Tbody(corpo da tabela) e o Tfoot(pé da tabela) da tabela, para finalizar também podemos colocar uma legenda na tabela com o Caption, o caption você só coloca texto, nada de tr ou td.

Efeito zebrado

O efeito zebrado é muito recomendado quando a tabela é grande, podemos fazer esse efeito bem rapido e fácil no CSS. Usamos a pseudo classe :nth-child(), entre parênteses podemos usar 3(três) valores, são odd(impar), even(par) ou (n°)n. O n a cada x(numero) ele vai aplicar o efeito.

Cabeçalho fixo em tabelas grandes

Para esse recurso, primeiro vamos escrever o caminho nas CSS thead > tr > th, segundo usamos a tag position com a propriedadestick; e por ultimo a tag top com 0px, mas lembrando de colocar a cor de fundo se não vai descer só o texto. Vale lembrar de colocar a propriedade position: relative no table, para o efeito funcionar.

Mesclagem de células

Atenção ao mesclar as células, pois os dois comando de mesclagem são parecidos e fácilmente de serem confundidos. O primeiro é rowspan que vai ser dentro de td, que fica td rowspan=" ", o rowspan vai pegar as linhas de baixo de acordo com o número que for colocado entre as aspas. O segundo é o colspan que também vai dentro de td, colspan=" " mas a diferença é que ele irá ocupar as linhas laterais. Veja o exemplo abaixo:

Exercício
Coluna A Coluna B Coluna C
1 2
4 5
8

Na coluna A o número 1, ela está ocupando dois espaços e para isso usamos o colspan para ocupar a coluna ao lado. Já os números 2 e 4 usamos o rowspan para ocupar as linhas abaixo.

Agrupando colunas

No HTML5 temos uma tag chamada colgroup que é utilizada com a tag col para fazer modificações nas colunas. Não esqueça de colocar o scope no col no começo da coluna para funcionar. Vale lembrar que a tag col não precisa de fechamento e pode utilar o span para a configuração pegar na proxima(s) coluna(s).

Tabelas responsiva

Para fazer uma tabela responsivaé bem facíl, basta colocar a tabela em uma div(id ou class você escolhe) e na CSS colocar o nome do id ou class, com o parametro overflow(y-x), lembre-se o (y) é vertical e o (x) horizontal. Pode escolher o overflow(transbordar) expecifico ou usar com os dois mas primeiro y e depois x. Ao usar a rolagem só vai ser criada no conteúdo da tabela.
