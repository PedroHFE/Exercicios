Condição

Condição é como se fosse uma bifurcação, os códigos começam a ser lido de cima para baixo em linha reta mas dependendo pode ser criar uma bifucação ou seja dependendo do resultado que obtivermos, aparecera/executar uma ação diferente e depois volta ao fluxo normal.

O primeiro que vamos aprender é o if que quer dizer 'si', o segundo é o else que quer dizer 'se não', exemplo: if (condição) {bloco 1 }else{bloco 2 }. A bifucação é lida mais ou menos assim, si o resultado for verdadeiro vai ser executado os comandos do bloco 1 se não (caso o resultado seja falso) o bloco 2 será executado.

Condição simples

A condição simples ela só tem um tipo de bloco, geralmente só o bloco verdadeiro. Por só ter o if e um bloco para executar se o resultado for verdadeiro, é chamado de condição simples. Exemplo: if (condição) { bloco 1}.

Condição composta

As condições compostas são aquelas formadas com if e else. Que executa um bloco de comandos de acordo com o resultado. Exemplo: if (condição) { bloco 1} else{ bloco 2}.

Condição Aninhada

Condições aninhadas são uma condição composta, com outras condições dentro dela, pode ser uma condição simples ou outras condições compostas. Exemplo: if (cond1){ bloco1 (caso for verdadeiro)} else{ }if (cond2){ bloco 2 (caso for verdadeiro)}else { bloco 3 (caso o bloco 1 e 2 derem falso)}.

Else if

Como em JavaScript é bem comum o uso de if e else, podemos pegar o if e jogar dentro do else. Exemplo: antes if (cond1){ bloco1} else{ if (cond2){ bloco 2} }
Exemplo: depois if (cond1){ bloco1} else if (cond2){bloco 2}
Ao fazer isso nós economizamos a escreita de um bloco aumentando a velocidade e praticidade, pode se criar um número ilimitado de else if dentro de uma estrutura.

Condição Multipla

A condição multipla serve para valores fixos. Não serve muito para intervalo de valores, ela tem a possibilidade não só do "sim" e "não", ela tem a possibilidade de outros valores, de volores fixos. Não é aplicavel a todo tipo de situação mas é muito útil em situações pontuais, específicas. Só funciona com números inteiros e com caracteres (com strings).

É representado da seguinte maneira, a gente tem um comando dentro do JS, que é o switch. O switch não condição e sim uma expressão, dentro do bloco colocamos vários valores, no final tem a cláusula do switch que é o default, é como se fosse o else do switch. Se nenhum dos valores forem satisfeitos, ele vai fazer o default que é o padrão. Importante para cada bloco, é preciso colocar o comando break.

Exemplo: switch(expressão) {
case valor 1:
brack
case valor 2:
break
casevalor 3:
break
default:
break }

O switch vai testar essa expressão para ver se ele bate com algun dos valores. Vamos supor que essa expressão resulte no valor 2, ele vai desviar automaticamente do valor 1, e vai começar a executar os comandos que estão dentro valor 2 e no final vai bater no break. Quando ele bate no break, o fluxo é desviado para fora do switch, caso não exista o break ele vai executar todos os comandos até achar um braek.
