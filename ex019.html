Aritméticos

Os operadores aritméticos são os sinais para fazer calculos que são +(adição), -(subtração), *(multiplicação), /(divisão real), %(divisão inteira) e **(potência). Todos esses operadores são chamados de operadores binários, pois precisam de 2 operandos. Diferenças entre a divisão real e a divisão inteira, a divisão real é divisão que tem o resultado quebrado (ex:5/2 = 2.5). Já a divisão inteira para antes de quebrar o resultado e o resultado é o número que sobra (ex:5%2 = 1). Lembre-se da mesma forma que na matemática existe precedência de operadores.

    Lista de Precedência
  1. ()
  2. **
  3. * / % caso tenha mais de um, seguira a ordem de chegada
  4. + - a ordem é lida esquerda para a direita

Auto-atribuições

Auto Atribuição: são atribuições à própria variável

Exemplo:
var n = 3
n = n ** 2
Então o resultado de n passa a ser o resultado do conta(16), isso é a auto-atribuição. Podemos simplificar quando a mesma variável recebe ela mesma, exemplo: n **= 2. Essa simplificação é aceita não só no JavaScript como também nas linguagens como Java, Python e PHP.

Podemos simplificar um pouco mais quando for adição(+) e subtração(-).
Exemplo:
var n = 5
n = n + 1 (R:6)
n += 1 (R:7)
n++ (R:8) ou n-- (R:6)
Com isso podemos simplificar ainda mais quando for adição e subtração, na hora de aumentar ou reduzir o valor em 1.

Relacionais

Para toda expressão que tenha um valor relacional ligada a ela, o resultado dessa expressão vai ser um valor booleano, do tipo Verdadeiro (true) ou Falso (false). Se usar operadores aritméticos na expressão primeiro resolve o aritmético e depois os relacionais mas o resultado será treu ou false.

  • > Maior
  • < Menor
  • >= Maior ou igual
  • <= Menor ou igual
  • == Igual
  • != Não igual
  • !== Desigual restrito
  • === Igualdade restrita (mesmo e o mesmo tipo)

    • Operadores relacionais não tem ordem de precedência.

Lógicos

A !(Negação) é tratado como operador unário, ou seja ele só tem um operando, depois da exclamação só trabalhamos com true, false e uma expressão que o resultado será verdadeiro ou falso. Exemplo: !true R:False (Uma coisa não verdadeira, é falsa), !false R:True (Uma coisa que não é falsa, é verdadeira).

A &&(Conjunção) é um operador binário, funciona com dois operadores lógicos um de cada lado. Exemplo: true && true R:true, true && false R:false, false && true R:false e false && false R:false.

A ||(Disjunção) é um operador binário, na disjunção basta só um resultado ser verdadeiro, para a resposta ser verdadeiro. Exemplo: true || true R:true, true || false R:true, false || true R:true e false || false R:true.

  1. ! Negação (Não)
  2. && Conjunção (E)
  3. || Disjunção (Ou)

    4. Já está na ordem de precedência.

Ternário

O operador ternário é dividido em 3 partes, ele usa 3 operandos. Usamos os símbolos de ? e :. Exemplo teste?true:false (dependendo do resultado do teste lógico ele vai retornar se é verdadeiro ou falso), média >= 7.0?"Aprovado":"Reprovado".
