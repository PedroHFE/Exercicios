DOM

O popularmente conhecido como DOM é a abreviação de Document Object Model, ele é a vertente Web do JavaScript pro qual a linguagem foi criada é uma poderosa ferramenta dentro do seu navegador, um conjunto de objetos JavaScript para tratamento de componentes visuais. Básicamente é um conjunto de objetos dentro do seu navegador que vai dar acesso aos componentes internos do seu website. O DOM não funciona dentro do node.js, ele está presente quando estou rodando JavaScript dentro do meu navegador.

Árvore DOM

Árvore DOM começa da raiz e essa raiz dentro do navegador é chamada de window (janela), então tudo dentro do JavaScript está dentro de um objeto chamado window (janela). Depois temos Locatio a localização do seu site (url), em seguida vem o Document o seu documento e History que grava a onde você estava e para onde você vai. Dentro de Document temos HTML que é pai de head e body, head que é pai de meta e title, body que é pai de h1, h2, p... E assim vai se formando uma árvore DOM. Podemos navegar dentro dessa árvore usando o DOM. Exemplo: window.document.write(window.document.URL)

Navegando pelos elementos

Podemos navegar pelos elementos de maneiras diferentes, elas são, por Marca, por ID, por Nome, por Classe e Seletor.

Marca

Por Marca fica assim getElementsByTagName(), lembre-se quem em JS as letras Maiúsculas fazem muita diferenças. Perceba que que Elements está no plural, por isso ele irá aplicar a todas as tags iguais. Exemplo: window.getElementsByTagName('p'), todo o comando que eu colocar a seguir irá afetar todos os p (paragráfos), caso eu queira afetar só o primeiro paragráfo posso sem dar espaço abrir e fechar [](conchetes) e dentro dele colocar um número, como queremos selecionar o primeiro p(paragráfo) entre os [0](conchetes) coloco o número 0, caso fosse o segundo paragráfo nós colocariamos o número 2.

ID

Já na opção de navegar por ID podemos criar um ID para o elemento a qual queremos e escrever assim: getElementById(). O ID é uma coisa única no código por isso a escreita do componente está no singular.

Nome

O Name é escrito no plural pois pode ter elementos com o mesmo nome, por isso em alguns casos você pode precisar usar os [](conchetes). Exemplo: getElementsByName().

Classe

Dá mesma maneira que a anterior, a Classe é plural por isso pode ser usada a varios elementos. Exemplo: getElementsByClasseName().

Seletor

O seletor é a forma mais recomendada pela maiorias dos manuais, utilizamos o querySelector() e o querySelectorAll(). O querySelector é a forma singular por assim dizer e o querySelectorAll é a forma plural, ambos aceitam a forma do CSS para encontar um elemento de #(para ID) e .(para classes).

Eventos DOM

Evento é tudo que possa acontecer com o elemento, geralmente o evento mais comum são os eventos de mouse, alguns são, mouseenter (quano o mouse entra dentro da área do elemento),depois vai disparar o proxímo evento mousemove (o mouse se movimentando dentro da área do elemento), outro evento é mousedown (quando clicam e seuguram), depois o mouseup (quando para de clicar), caso seja um clique rápido é chamado de click e quando o mouse saí da área do elemento é chamado de mouseout. Existe centenas de outros eventos em DOM.

Funções

Uma função em JavaScript, é conjunto de códigos(linhas) que só vão acontecer quando o evento ocorrer. O código fica dentro de um bloco e essas linhas de códigos só serão disparadas atráves de um evento, bloco em JS é delimitado entre os sinais de {}(chaves), antes de abrir o bloco devemos colocar a palavra function e em seguida dar um nome para essa função, geralmente damos o nome de acordo com a ação que será executada, funções sem nomes são conhecidas como anônimas. Exemplo: function ação(). Dentro desse () (chaves) podemos colocar alguns parâmetros, pode ser um ou mais.

Disparando eventos em HTML

Podemos dispara eventos no HTML colocando dentro da tag a palavra on e o nome do método, em seguida entre ""(aspas duplas) o nome da ação e () (chaves). Exemplo: <div id="area" onclick="clicar()"

Disparando eventos no JavaScript

Temos um método no JavaScript chamado de listeners, que são ouvidores, eles ficam prestando atenção dentro do próprio JS. O listeners só podemos criar um evento por vez, começa assim .addEventListener, depois abre () parenteses, coloca entre ''(aspas simples) o nome do evento e depois o nome da ação. Exemplo: .addEventListener('click', clicar), não tem parenteses no nome da ação.
