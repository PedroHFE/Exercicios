Titulos, paragrafos e quebra de linhas

Tag para titulos é </h1>.

Sub titulos são h2,h3,h4 h5 e h6.

Para criar um paragrafo só uasar a tag</p>

Para quebrar o texto só usar </br>

Semântica na HTML5

O significado dos vocabulos, por oposição à sua for.

Simplificando é: O significado é tem mais valor que a sua forma.

Não usar tags obsoletas. Verificar no site oficial W3c.

Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em destaque usando a tag B (não semântica)

Nesta frase, temos um termo em destaque usando a tag STRONG (Semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (Não semântica)

Nesta frase, temos um termo em itálico usando a tag EM (Semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK. (Semântica)

Texto grande e texto pequeno

Neste paragrafo, vamos deixar um texto grande usando a tag BIG. (Obsoleta)

Neste paragrafo, vamos deixar um texto pequeno usando a tag SMALL. (Semântica)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag DEL (Semântica)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag INS. (Semântica)

EXiste também o sublinhado, usando a tag U (Não Semântica)

Texto sobrescrito

Para inserir coisas do tipo:, x20+3. Usando a tag SUP (semântica)

Texto subscrito

Para inserir coisas do tipo H2O. Uando a tag SUB (Semântica)
