Tamanhos de Fontes

Por padrão a W3C recomenda usar a medida px ou em. Noralmente a maioria dos sites, por padrão, o tamanho da fonte é 16px. Geralmente 1 em é o tamanho de 16px: 1em = 16px.

Medidas Absolutas

As medidas Absolutas não são muito recomendadas, pois a exibição de cada tela é diferente.

  • Centímetro (cm)
  • Milímetro (mm)
  • Polegada (in)
  • Pixel (px)
  • Ponto (pt)
  • Paica (pc)

Medidas Relativas

  • Relativo ao tamanho do M (em)
  • Relativo ao tamanho do x (ex)
  • Relativo a fonte configurada no body (rem)
  • Relativo a largura da viewport, view width (vw)
  • Relativo a altura da viewport, view height (vh)

Pesos da fontes

Não confunda as palvras na hora de escrever o codigo, as mais comuns de se confundir são Weight (peso), Width (Largura) e height (Altura). algumas fontes tem mais pesos que outras.

  • Lighter
  • Normal
  • Bold
  • Bolder
  • Numerica 100 a 900

Declaração de fontes

Como usamos muito a as tags font-family, font-weight, font-size e font-style. Acaba tomando muito tempo e espaço, por isso podemos abreviar tudo isso em uma unica linha de codigo. Isso é chamado de Shorthand. Atenção usar na ordem correta!. A ordem correta é a segunite font-style -> font-weigth -> font-size -> font-family.

O codigo fica assim font: italic bolder 1em 'Work Sans', sans-serif;

Não use muitas fontes no seu site, o padrão e mais comum são de 2 a 4 familias de fontes diferentes. Uma ótima ferramenta é o google fonts. Sempre que for importar uma fonte do google fonts usar a opção @import no começo de um documento CSS separado ou logo abaixo do style caso seja CSS interno.

Outro site bem util para adquirir fontes externas(gratuitas) é o dafont, sempre é bom lembra de verificar se a fonte tem glifos (acentuação).

Ao usar fontes personalizadas utilize a regra @font-face, para melhor organização faça uma pasta separada, para deixar as fontes.
