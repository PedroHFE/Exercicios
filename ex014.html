O que é um iframe?

Os frames já estão muito ultrapassados mas os iframes não, esse 'i' do começo da palavra é de 'inline', basicamente são quadro em linha, no w3c só chama de iframe não tem tradução para o português.

O iframe é um quadro na mesma linha, que parece uma 'nova guia do navegador' exibindo um site, mas atenção, alguns sites tem proteções contra o seu conteúdo ser exibidos por iframes.

Nem todos os navegadores são compatíveis com iframes dá para colocar uma frase ou links no meio para aparecer caso o navegador não tenha suporte para o iframe.

Por padrão o iframe vem com o tamanho de 300px de largura e 150px de altura, claro que esses valores podem ser mudados dentro da tag mas os parâmetros das CSS é os que prevalecem.
