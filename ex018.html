Tratamento de dados

Antes de começarmos, só para lembrar que o sinal de mais(+) tem a função concatenação, quando o tipo for string, ele só vai fazer a adição se for do tipo number. O comando prompt retorna o valor como uma string, independente do que foi digitado.

Para fazer essa interação usamos o comando prompt para perguntar o nome e o alert para dar o resultado e criamos uma variável pra guardar o dado(nome) recebido e usá-lo no alert.
<script >
var nome = window.prompt("Qual é o seu nome?")
window.alert('Prazer em te conhecer ' + nome + '!')
</script >

Atualmente não é mais necessário colocar window antes do comando, nem o ponto e vírgula(;) no final. Para a mensagem aparecer não pode esquecer de usar aspas (' ' ou " ").

<script >
var n1 = window.prompt('Digite um número')
var n2 = window.prompt('Digite outro número')
var r = n1 + n2
window.alert("A adição dos números foi de " + r + "!")
</script >

Ao deixar igual ao exemplo acima, o comando só irá juntar os valores ex: 10 + 0 apareceria 100 em vez de 10. Para concertar isso tem que converter o n1 e o n2 para números.

Podemos fazer a conversão antes dos dados ir para as variáveis (n1 e n2) ou depois. Para fazer a conversão temos 3 maneiras. A primerira é o Number.parseInt(n) que vai converter para números inteiros, lembrando que o N de number e o I de int tem que ser escritos em maiúsculos. A segunda maneira é usando o Number.parseFloat(n) que irá converter para números reais(números quebrados) e igual ao maneira anterior tem obrigação de letras maiúsculas nesse caso o N de number e o F de float.

A última maneira é escrever Number(n) e o próprio JS vai descidir se o valor é inteiro ou real. Essa é maneira mais utilizada mas as vezes pode ocorrer de precisar forçar a leitura de um número real ou inteiro. É obrigatório o uso de parenteses, pode usar as 3 maneiras mas a última é mais eficaz.

Exemplo:
var n1 = Number.parseInt(window.prompt('Digite um número')) ou var r = Nunber.paseInt(n1)
var n2 = Number.parseFloat(window.prompt('Digite outro número')) ou var r = Number.parseFloat(n2)
var n1 = Number(window.prompt('Digite um número')) ou var r = Number(n1) + Number(n2)

Podemos também fazer o contrário, converter um Number para String, existem duas maneiras String( n) e n.toString(). Não esqueça das letras maiúsculas.Ex:
window.prompt('A adição do números foi de ' + String(r) + '!') ou window.prompt('A adição do números foi de ' r.toString() + '!')

Formatando Strings

Ao escrever declarações o Placeholder ${} ajuda muito a ganhar tempo e deixa o código mais limpo, mas é obrigatórioo uso de crazes, ele não funciona com aspas simples e nem aspas duplas. Para Strings simples sem interpolação usamos aspas simples/duplas e Strings com interpolação usamos Template Strings para utilizar o placeholder.

var s = 'JavaScript'
'Eu estou aprendendo s' não aparece o nome JavaScript // não faz interpolação
'Eu estou aprendendo' + s já aparece o nome JavaScript //usa concatenação
`Eu estou aprendendo ${s}` já aparece o nome JavaScript // usa template string
s.length // quantos caracteres a string tem
s.toUpperCase() // tudo para letras 'MAIÚSCULAS'
s.toLowerCase() // tudo para letras 'minúsculas'

Formatando Números

Algumas formatações de números, primeriro para colocar casas decimal a mais, o segundo a trocar o ponto(.) por virgula(,) e último transforma o valor no formato de salário, para mudar a moeda do país mude o currency 'BRL'. Da mesma maneira que o string tem vários comando de formatação o números também.

var n1 = 1542.5
R: 1542.5
n1.toFixed(2)
R: 1542.50
n1.toFixed(2).replace('.' , ',')
R:1542,50
n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})
R: R$ 1,542,50
