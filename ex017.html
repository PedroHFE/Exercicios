JavaScript

Variáveis

No javascript, além de utilizar a palavra var, também pode usar a palavra let e o nome const. O sinal de =(igual) é lido como recebe. Quando se coloca um valores dentro de uma variável é chamado de atribuição, quando você atribui uma cadeia de caracteres é conhecido como string, é aceito 3 tipos de aspas que são aceitas em JS, são: ""(aspas duplas), ''(aspas simples) e ``(crases). Existe diferenças para o JS de cada uma mas veremos mais para frente.

Os nomes das variáveis são chamados de identificadores, egistem algumas regras para os nomes dos identificadores. Elas são:

  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Podem começar com $ ou _
  • Não podem começar com números
  • Não podem conter espaços
  • Não podem ser palavras reservadas (Ex: Function, var, let...)

Dicas para criar nomes de variáveis.

  • Maiúsculas e Minúsculas fazem diferença
  • Tente escolher nomes coerentes para as variáveis
  • Evite se tornar um programador alfabeto(a,b,c,d...) ou um programador contador(n1,n2.n3,...)

Tipos primitivos (primordiais)

Number

O JS não faz diferenciasão de números mas existe diferenças entre números inteiros (-15, 8, 31) que não são fracionados e os números reais chamados de float, outras linguagens de programação fazem diferenciasão.

String

São cadeias de caracteres, que podem ser um conjunto de números por exemplo o CPF,RG, Telefone... pois eles tem traço, ponto e etc.

Boolean

True (verdadeiro) e False (falso), são conhecidos como valores booleanos.

Para trabalhar com os Data Types(tipos primitivos) usamos o comando typeos(tipo de). Pode se criar uma variálvel e pedir o typeof(tipo de) dela, como também colocar o valor literal, o JS chama de literal quer que é literalmente. Quando se cria uma variálvel você não define um tipo primitivo, você diz que ela é uma variálvel, é um espaço. Pode aumentar e diminuir conforme o programa for rodando.
