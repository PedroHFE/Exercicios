Tipos de Seletores

O primeiro tipo de seletor é o id (identication), toda tag que tiver em HTML id, em CSS é #. A regra do W3C é, não usar mais de um id com nome igual no mesmo documento. Também é bom lembrar que o sinal de maior(>) é chamado de clildren(filho).

O segundo tipo de seltor é class (classe), o class é para ter vários elementos com o mesmo nome em um documento. Toda tag que em HTML é class, nas CSS são ponto(.), evite dar nomes as suas classes pela forma mas sim pela funcionalidade da mesma. É possível o mesmo elemento ter mais de uma classe.

Pseudos Classes

Uma pseudo-classe CSS é uma palavra-chave adicionada às declarações de um seletor após um sinal de dois pontos(:) e especificam um estado especial de um elemento.Existem várias pseudo-classes algumas são.

  • :Hover
  • :Visited
  • :Active
  • :Checked
  • :Empty
  • :Focus

Pseudos Elementos

Já um pseudo-elemento CSS é uma palavra-chave adicionada às declarações de um seletor após dois sinais de dois pontos(:) e permitem que você formate um pedaço específico do elemento referemciado. Os principais pseudo-elementos são.

  • ::Before
  • ::After
  • ::First-letter
  • ::First-line
