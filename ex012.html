Trabalhando com imagens de fundo

Para utilizar imagens no fundo é fácil, basta usar a tag background-image: url() e colocar o caminho da imagem. Colocar a imagem é fácil, mas não pode esquecer que por padrão a imagem vai carregar na altura e largura do arquivo e também vai se repetir.

Tags para configuirar a imagem

  1. background-size: 100px Largura(Width) 100px Altura(heigth);
  2. background-repeat: repeat; Padrão que repete a imagem
  3. background-repeat: no-repeat; Não repete a imagem
  4. background-repeat: repeat-x; Repete a imagem na horizontal
  5. background-repeat:repeat-y; Repete a imagem na vertical
