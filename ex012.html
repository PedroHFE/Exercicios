Trabalhando com imagens de fundo

Para utilizar imagens no fundo é fácil, basta usar a tag background-image: url() e colocar o caminho da imagem. Colocar a imagem é fácil, mas não pode esquecer que por padrão a imagem vai carregar na altura e largura do arquivo e também vai se repetir.

Tags para configurar a imagem

    Background size e background repeat

  • background-repeat: repeat; Padrão que repete a imagem
  • background-repeat: no-repeat; Não repete a imagem
  • background-repeat: repeat-x; Repete a imagem na horizontal
  • background-repeat:repeat-y; Repete a imagem na vertical
  • background-size: 100px Largura(Width) 100px Altura(heigth);

Além de escolher o nível de repetição do background, também podemos mudar a posição de referência de início das repetições. Por padrão, é considerado o canto esquerdo superior (left top), mas podemos ter várias opções. Use a imagem abaixo como referência sempre que precisar definir a posição do fundo com a propriedade background-position no seu código.

Outra coisa que podemos fazer é redimensionar a imagem para forçá-la a caber na caixa. Por padrão, nenhum redimensionamento será aplicado, e a imagem será exibida do seu tamanho matural. Porém, podemos usar a propriedade background-size para alterar esse comportamento.

    Os valores aceitos por essa propriedade são:
  • auto (padrão) a imagem de fundo será aplicada em seu tamanho original.
  • [length]px Redimensiona a largura da imagem e faz a altura se adaptar automaticamente.
  • [length]% Podemos informar as duas dimensões na sequência ou também usar valores percentuais.
  • contain Muda o tamanho da imagem para que seja sempre totalmente exibida na tela, sem nenhum corte.
  • cover Redimensiona a imagem para que cubra o contêiner, mesmo que para isso ocorram alguns cortes.

Background-attachmente

A última propriedade que podemos configuirar é o vínculo (attachment) da imagem de fundo com o resto do documento, principalmente se o conteúdo for mair do que a altura da página e seja necessaário vazar uma rolagem vertical.

    A propriedade background-attachment aceita os valores:
  • scroll (padrão) a imagem de fundo vai rolar junto com o conteúdo.
  • fixed A imagem de fundo vai ficar fixada enquanto o conteúdo vai sendo rolado.

Existe uma shorthand para simplificaar o uso de propriedades que se apliquem ao fundo de uma caixa. A propriedade abreviada background pode ser declarada agrupando as seguintes configurações:

  1. background-color: black;
  2. background-image: url('endereco/imagem.jpeg');
  3. background-position: center center;
  4. background-repeat: no-repeat;
  5. background-size: cover;
  6. background-attachment: fixed;

A declaração ficaria sim background: black url('endereco/imagem.jpeg') center center no-repeat cover fixed; Na documentação do W3C está exatamente assim, mas pode ocorrer um erro na parte do size, ter que utilizar a shorthand background sem o size e criar em baixo um background-size: cover;.

Centralização vertical em contêineres

Contêiner tem um significado simples e direto: "aquele que contém coisas". Aprendemos que existem elementos que podem conter outros elementos. As <div> são um exemplo de elemento contêiner. Quando queremos centralizar blocos horizontalmente, aprendemos a usar o margin:auto ; nas folhas de estilo(CSS). Agora vamos aprender a centralizar na vertical.

Para começar temos que criar um contêiner uma hierarquia simples de dois blocos, vou criar duas <div> ,dar um id com o nome de 'fora' e a outra com id 'dentro'.

O contêiner(um box) por padrão já vem com um posicionamento relativo no HTML e conteúdo com um posicionamento absoluto. Para configurar é preciso deixar isso explicito com a tag position: relative; para a div de fora e position: absolute; para a div de dentro. Quando definimos um bloco com posicionamento absoluto, podemos personalizar a sua posição exata através das propriedades left e top. Vamos configurar os dois na metade do contêiner.

Note que, o retângulo interno foi realmente posicionado a 50% da tela, mas pelo canto superior esquerdo da caixa. Vamos realizar uma tranformação e mover o retãngulo interno para a esquerda e para cima, assim ele ficará efetivamente centralizado com a tag transform: translate (-50%, -50%);.

Pronto! Essa é apenas uma das técnicas de centralização de conteúdo, mas as outras requerem aprender outros conceitos mais aprofundados das CSS (folhas de estilos), como as caixas flexíveis (Flexbox).
