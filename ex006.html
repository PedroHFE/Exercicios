Tipos de estilos em CSS

Estilo Inline

Inline significa na mesma linha, todo CSS vai ser feito na mesma linha dentro das < >, só é recomendado o CSS Inline em algumas correções expecificas.

Estlo Interno / Locais

A vantagem do estilo interno é que você pode usar a tag STYLE, abrir chaves {} que se
chama Seletor e tudo que estiver em {} é chamado de Declaração (Conjunto de Propriedade e Valor).
Se for só um documento pode utilzar sem problma.

Estilos Externos

É a melhor e mais eficiente maneira de usar CSS, iremos criar um link para o CSS de um
arquivo excluso para o desing Geralmente o estilo acabam tendo mais linhas de codigo do
que o conteudo. Os 3 estilos podem ser usados juntos e pode também usar mais de um link CSS.
